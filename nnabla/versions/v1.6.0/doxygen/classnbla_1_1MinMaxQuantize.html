Detailed Description

template<typename T>
class nbla::MinMaxQuantize< T >

MinMaxQuantize quantizes values in integer representation.

Inputs:

  • N-D array of input
  • N-D array of minimum quantization range (modified during forward execution)
  • N-D array of maximum quantization range (modified during forward execution)
  • N-D array of minimum quantization level
  • N-D array of maximum quantization level execution)
Template Parameters
TData type for computation.
Parameters
decayDecay rate for the exponential moving average.
x_min_maxUse the min and max of x to compute quantization ranges.
emaUse the exponential moving average for the min and max quantization. ranges.
ste_fine_grainedStraight Through Estimator is fine-grained or not.
