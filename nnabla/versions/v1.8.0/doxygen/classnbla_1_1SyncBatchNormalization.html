Detailed Description

template<typename T>
class nbla::SyncBatchNormalization< T >

Batch normalization with sync between other processes at training time defined as

\[ \begin{array}{lcl} \mu &=& \frac{1}{M} \sum x_i\\ \sigma^2 &=& \frac{1}{M} \left(\sum x_i - \mu\right)^2\\ \hat{x}_i &=& \frac{x_i - \mu}{\sqrt{\sigma^2 + \epsilon}} \\ y_i &=& \hat{x}_i \gamma + \beta. \end{array} \]

.

Inputs:

  • N-D array of input.
  • N-D array of beta which is learned.
  • N-D array of gamma which is learned.
  • N-D array of running mean (modified during forward execution).
  • N-D array of running variance (modified during forward execution).

Outputs (1 or 3):

  • N-D array.
  • (Optional) N-D array of batch mean.
  • (Optional) N-D array of batch variance.
Template Parameters
TData type for computation.
Parameters
commThe communicator
groupThe name of the communicator group
axesAxes mean and variance are taken.
decay_rateDecay rate of running mean and variance.
epsTiny value to avoid zero division by std.
See also
Implementing Synchronized Multi-GPU Batch Normalization https://hangzhang.org/PyTorch-Encoding/notes/syncbn.html
