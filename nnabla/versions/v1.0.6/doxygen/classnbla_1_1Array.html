Detailed Description

An abstract class of Array interface.Shared pointer of NdArray.

This is extended to implement a new array class (see CpuArray, CudaArray etc.).

NOTE for developers: Do not directly access object_ from anywhere except `allocate` and `deallocate` functions. Use `pointer<T>()` or `const_pointer<T>()`. Objects are lazily allocated when they are called.
