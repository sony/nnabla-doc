Detailed Description

template<typename T>
class nbla::RandomErase< T >

RandomErase randomly erase the inputs and replace with random values.

Inputs:

  • N-D array of three or more dimensions

Outputs:

  • N-D array.
Template Parameters
TData type for computation.
Parameters
probProbability to erase.
area_ratiosLow and high of the area ratio range.
aspect_ratiosLow and high of the aspect ratios range.
replacementsLow and high of the replacement value range.
nMax number of patches to be erased.
shareUse a same bounding box randomly picked over the feature dimension when being True. Default is False.
inplaceThe output array is shared with the input array if True.
base_axis
seedRandom seed. When -1, seed is sampled from global random number generator.
channel_lastIf True, the last dimension is considered as channel dimension, a.k.a NHWC order.
ste_fine_grainedStraight Through Estimator is fine-grained or not. Default
