Graph Definition¶

In NNabla, graph_def represents a kind of representation of a computation graph which is special designed for optimization storage and format converter.

NNabla provides a group of functions to allow load or save a computation graph from or to storage, manipulate computation graph directly on such graph_def format, and all necessary converters between various neural network frameworks.
