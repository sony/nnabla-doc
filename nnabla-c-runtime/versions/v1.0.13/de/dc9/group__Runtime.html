Detailed Description

Calculate feed forward neural network with Internal network representation.

  • Sequence to use Runtime API.
    • Initialization
      • Initialize runtime context with rt_initialize_context()
      • Allocate I/O buffers in size get from rt_num_of_input(), rt_input_size(), rt_num_of_output() and rt_output_size()
    • Diagram
    • Inference
      • Prepare input data and buffer to output.
      • rt_forward()
      • Use output data in output buffer.
      Runtime APIs provides following functions.
  • rt_allocate_context()
  • rt_initialize_context()
  • rt_free_context()
  • rt_num_of_input()
  • rt_input_size()
  • rt_input_dimension()
  • rt_input_shape()
  • rt_num_of_output()
  • rt_output_size()
  • rt_output_dimension()
  • rt_output_shape()
  • rt_forward()
