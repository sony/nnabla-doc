Update function info

NNabla C Runtime is part of NNabla but source code of this is independent from NNabla code.

If you want to follow newest NNabla by hand, you can update functions.yaml with following command.

$ nnabla_cli function_info -o build-tools/code-generator/functions.yaml

Compile

Otherwise, you can compile NNabla C Runtime just typing make .

$ make
